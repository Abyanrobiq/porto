PROJECT IMAGE

About The Project

Project ini merupakan aplikasi Point of Sales yang dikembangkan untuk membantu proses operasional Kedai Kopi Ampuh.

Sistem ini menyediakan fitur transaksi penjualan, pengelolaan produk, pengelolaan stok, pembelian, resep produk, dan laporan keuangan.

Technologies

Laravel PHP MySQL Filament HTML CSS JavaScript

Key Features

Point of Sales

Membantu kasir melakukan proses transaksi penjualan dengan lebih cepat dan efisien.

Stock Management

Mengelola stok bahan baku dan produk secara lebih terstruktur.

Financial Reports

Menyediakan laporan keuangan seperti jurnal umum, buku besar, dan laba rugi.
